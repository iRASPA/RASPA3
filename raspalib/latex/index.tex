\doxyparagraph*{This software is a general purpose classical simulation package.}



It has been developed at the University of Amsterdam (Amsterdam, The Netherlands) during 2022/2026 in active collaboration with Eindhoven University of Technology (Eindhoven, Netherlands), Delft University of Technology (Delft, The Netherlands), and Northwestern University (Evanston, USA).

\doxysection*{Implemented so far}


\begin{DoxyItemize}
\item rigid molecules and rigid framework
\item multiple systems, box or framework
\item grand canonical ensemble (CBMC, CFCMC, and CB/\+CFCMC)
\item Gibbs ensemble (CBMC, CFCMC, and CB/\+CFCMC)
\item Monte Carlo NPT ensemble
\item transition matrix Monte Carlo
\item Molecular Dynamics NVT ensemble (Nose-\/\+Hoover thermostat)
\item binary restart
\item blocking pockets
\item PDB-\/movies, energy histograms, number of molecule histograms
\item RDF, MSD-\/order-\/N, VACF, density grids (cube files)
\item charge equilibration
\item MC/\+MD hybrid move
\item tail-\/corrections for CFCMC
\item grids for rigid frameworks
\item writing and reading restart-\/files
\item external field interpolation grids
\end{DoxyItemize}
