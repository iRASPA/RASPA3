A pressure input of 10 Pascal in the input file is converted to \textquotesingle{}internal units\textquotesingle{} by dividing by $1.66054 \times 10^7$. In the output, any internal pressure is printed, multiplied by $1.66054 \times 10^7$. It is not necessary to convert units besides input and output, with a few exceptions.

One of the exceptions is the Coulombic conversion factor\+: \$\$\textbackslash{}frac\{q\+\_\+i q\+\_\+j\}\{4\textbackslash{}pi \textbackslash{}epsilon\+\_\+0\}=\textbackslash{}frac\{\textbackslash{}text\{charge\}\texorpdfstring{$^\wedge$}{\string^}2\}\{4 \textbackslash{}pi \textbackslash{}times \textbackslash{}text\{electric constant\} \textbackslash{}times \textbackslash{}text\{length\} \textbackslash{}times \textbackslash{}text\{energy\}\} = 138935.\+4834964017\$\$ where the electric constant is $8.8541878176 \times 10^{-12}$ in units of $\text{C}^2/(\text{N} \cdot \text{m}^2)$. This factor is needed to convert the electrostatic energy to the internal units at every evaluation.

The Boltzmann constant $k_B$ is\+: \$\$k\+\_\+B = \textbackslash{}text\{Boltzmann constant\}/\textbackslash{}text\{energy\} = 0.\+8314464919\$\$ with the Boltzmann constant as $1.380650324 \times 10^{-23}$ in units of J/K, and $k_B = 0.8314464919$ in internal units.


\begin{DoxyItemize}
\item Numbering is based on the {\ttfamily C}-\/convention, i.\+e. starting from zero.
\item Files in the current directory always have preference. Sometimes one would like to try various parameters for force field fitting, for example. In order to avoid making a lot of directories for each force field, it is more convenient to have the {\ttfamily force\+\_\+field.\+def} file in the {\itshape current} directory. 
\end{DoxyItemize}
