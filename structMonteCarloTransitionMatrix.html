Detailed Description

Performs Monte Carlo simulations for molecular systems.

The MonteCarloTransitionMatrix struct orchestrates the execution of Monte Carlo simulations, including initialization, equilibration, and production stages. It manages multiple systems, random number generation, simulation parameters, and output generation.
